# Sistema-de-Gest-o-Escolar-SisGESC-
O SisGESC é um banco de dados para gestão escolar que abrange cadastros de alunos, professores e turmas, além de matrículas, notas e frequência. Inclui DER, Dicionário de Dados e Regras de Negócio para garantir integridade. Suporta BI e IA para análises preditivas e decisões estratégicas na educação.
